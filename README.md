# TEDxICEAS - Shaping Us

Welcome to the official website for **TEDxICEAS**. This project is built using modern web technologies to provide a high-performance, responsive, and visually stunning experience.

## Tech Stack

- **Vite** - For fast development and optimized builds.
- **React** - Component-based UI library.
- **TypeScript** - For type-safe and maintainable code.
- **Tailwind CSS** - For modern, utility-first styling.
- **shadcn-ui** - Beautifully designed components.
- **Framer Motion** - For fluid and premium animations.

## Getting Started

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) and `npm` installed.

### Installation

1. Clone the repository:
   ```sh
   git clone <repository-url>
   ```
2. Navigate to the project folder:
   ```sh
   cd shaping-us-website
   ```
3. Install dependencies:
   ```sh
   npm install
   ```

### Development

Run the development server:
```sh
npm run dev
```

### Build

Create a production-ready build:
```sh
npm run build
```
